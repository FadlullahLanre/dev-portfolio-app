# Simple Developer Portfolio Application (Zuri internship)

# Description

*This is a Zuri Internship Task for the backend track (NodeJs)*

A portfolio web application with home, about,and contact HTML pages.

**Home Page** : The homepage should contain your data (bio)

**About**: The about me page should contain brief information about you

**Contact**: The contact page should contain links to your social media handles
 
# Instructions to use the app

1. Create a server using the **HTTP module**

2. Create routes that render the HTML pages you created (Home, About, Contact) on a browser.

3. The base URL of your app should render the homepage, there should be a /home route that redirects to the base of your application.


**N.B**: 

1. No third-party dependencies should be used.
